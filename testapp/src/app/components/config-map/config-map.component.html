Map configuration

Description

Shows map configuration possibilities.

Example

Code

        
          <bric-map [config]="config" [layers]="layers"></bric-map>

Properties

Property name Description Possible values
minZoom The min zoom level. Defines how much the user can zoom out. Any number. Default is 8.
maxZoom The max zoom level. Defines how much the user can zoom in. Any number. Default is 20.
zoom The default zoom level. Will be set at the map initialization. Any number between min and max zoom. Default is 14.
projection The projection to use for the map. A valid projection. Default is EPSG:31370.
center The default coordinates where the map will be centered at initialization. An array of 2 numbers. Default is [149656, 169933.8] (more or less the center of Brussels.
scaleLine Flag to control the display of the scaleline on the map. Boolean. Default is false.
overviewMap Property allowing to display a small map on the bottom left corner to allow the user to localize himself on the map when zooming. Default is null (overview map is not displayed). If the id of one of the layers is used, the overviewmap is displayed and uses that layer.

Notes

Configuration properties can be changed at execution except for center and zoom as these 2 properties will be changed by the user when he navigates.
