WFS layers

Description

Shows a map with WFS layers (data layers).

Example

Event : 

  





  
Code

  
    
      
        
          <bric-map [config]="config" [layers]="layers"></bric-map>
        
      

    
    
      
        
      

    
  




  
Properties

  
Layers properties are similar to WMS layers. Only extra properties are shown here.

  

    
      

        Property name
        Description
        Possible values
      

    
    

      type
      The type of layer
      
        For custom layers only. Defines if the layer is a WMS (pictures) or a WFS (data). Possible values are:
        
    
          
  • BricMapLayerType.wms
    • 
          
  • BricMapLayerType.wfs
    • 
        

      		
    

    

      styles
      Define the layer style
      A valid style object. See the styling example
    

    

      cluster
      Activate or deactivate the clustering
      Boolean, default is false. See the cluster example
    

    

      selectable
      Activate or deactivate the feature selection
      Boolean, default is false.
    

  





  
Events

  
If selectable is set to true, an event is triggered when a feature is selected or unselected.

  
Usage : 

  
    
  <bric-map [config]="config" [layers]="layers" (mapEvent)="dealWithEvent($event)"></bric-map>
    
  

  
The event is an object with the following properties

  

    
      

        Property name
        Description
      

      

        feature
        An ol.Feature object that represents the selected/unselected feature.
      

      

        layer
        The layer id on witch the action has occured.
      

      

        action
        The action type. Can be select if the feature has been selected or unselect if the feature has been unselected.
      

    
  





  
Notes

  
The selectable flag allows selection of only one feature at the time. It's not possible to select more than one.

  
If a feature is already selected and the user selects another one, the unselect event is sent first, then the select event is sent for the new featire.

  
The selectable attribute can be changed at anyt time.
