BricGis Module

Description

BricGisModule is an angular2 component that simplify maps creation.

It allows to easily add on an application :

  • A map with different types of layers and interactions.
  • A search address componant that allows the user to look for an address. It provides auto-completion.
  • A table of content component that can be linked to the map to control layers transparency and visibility.

Installation

npm install bric-gis --save

Configuration

Styling

In order to display the map correctly, the ol CSS must be added to the application.

The CSS file can be found in node_modules/openlayers/dist/ol.css
